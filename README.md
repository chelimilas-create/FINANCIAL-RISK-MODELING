# Financial Risk Analysis using Python

## Project Overview

This project analyzes the German Credit dataset to understand customer borrowing behavior and identify financial risk patterns. The analysis focuses on loan purposes, credit amounts, loan durations, and housing status using Python for data cleaning and visualization.

## Objectives
Analyze customer borrowing patterns.

Identify the most common loan purposes.

Study the relationship between credit amount and loan duration.

Compare average credit amounts across housing types.

## Key Business Insights
Car loans are the most common loan type among customers.

Larger loan amounts are generally associated with longer repayment periods.

Loan duration varies depending on the loan purpose.

Most customers borrow small to medium-sized loan amounts.

Customers with free housing have the highest average credit amount.

## Business Recommendations
Review high-value loan applications carefully.

Offer flexible repayment plans for long-duration loans.

Develop loan products based on customer borrowing trends.

Monitor customer segments with higher average borrowing.

Regularly analyze lending patterns to support better credit decisions.


Conclusion

This project demonstrates how Python can be used to clean, analyze, and visualize financial data to uncover customer borrowing patterns and support data-driven lending decisions.
